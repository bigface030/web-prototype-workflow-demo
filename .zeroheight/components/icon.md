## Principles
Icons should allow users to quickly identify content and accurately convey meaning.
-   **Clarity**: Avoid excessive detail.
-   **Conciseness**: Simple shapes are better.

## Categories
1.  **Web Icons**:
    -   Used for Website or App interfaces.
    -   Functional icons (Search, Close, Menu, etc.).
2.  **Marketing Icons**:
    -   Used for PPTs, printed materials, and ads.
    -   More illustrative/detailed.
3.  **Accommodation Facilities**:
    -   Specific set for representing room amenities (WiFi, Pool, etc.).

## Usage
-   **Sizing**: Use standard grid sizes (e.g. 16px, 24px, 32px).
-   **Color**: Inherit text color or use specific system numbering.
