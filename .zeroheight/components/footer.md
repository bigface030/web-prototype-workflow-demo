# Footer

**Link**: [Footer Documentation](https://zeroheight.com/8ei12wwuw/p/653285-button) (Note: Verify URL, extracted via Search)

## Usage
Placed at the bottom of pages. Contains navigation, legal info, and contact details.

## Variations
1.  **Full Version**: Includes links grid, social icons, app download.
2.  **Copyright Only**: Minimal version for checkout/simple pages.

## Layout & Breakpoints
-   **Desktop (>=1280px)**:
    -   4-column layout.
    -   Content Max-Width: 1280px.
    -   Margins: 32px side.
-   **Tablet (992px - 1279px)**:
    -   Single column layout often (or collapsed links).
    -   Width: 100%.
-   **Mobile (<768px)**:
    -   Single column vertical stack.
    -   No hover effects.
    -   Margins: 16px (or 8px).

## Specs
-   **Margin Top**: 80px (distance from page content).
-   **Content**:
    -   "About Us", "Customer Service", "AsiaYo", "Download App".
    -   Payment Methods icons.
    -   Social Media icons.
    -   Copyright text + Legal links (Privacy Policy, Terms).

## Dos & Don'ts
-   **Do**: maintain consistent padding across pages.
-   **Don't**: Remove the footer on main pages. Use "Copyright Only" for focused funnels (Checkout).
