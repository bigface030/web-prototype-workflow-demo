# Tag

**Link**: [Tag Documentation](https://zeroheight.com/8ei12wwuw/p/653285-button) (Note: Verify URL, extracted via Search)

## Usage
Categorize items or show product attributes.

## Styles
1.  **Outline**: Border + Light Background.
2.  **Solid**:
    -   Dark Background + White Text.
    -   Light Background + Dark Text.

## Anatomy
-   **Container**: Shape wrapper.
-   **Label**: Concise text.
-   **Icon (Optional)**: Support for visual recognition.

## Dos & Don'ts
-   **Do**: Use for simple categories.
-   **Don't**: Use **Badge** for unread counts or status alerts.
