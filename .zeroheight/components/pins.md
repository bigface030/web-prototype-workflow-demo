# Pins

## Usage
Map markers for property locations and pricing.

## Types
1.  **Single Pin**:
    - Shows specific price (e.g., "$1,200").
    - Currency symbol + Amount (comma separated).
2.  **Multiple Pin** (Cluster Representative):
    - Shows price range (e.g., "$1,200 ~").
    - Includes **Badge**: Number of properties in this group.

## Anatomy
- **Container**: Rounded rectangle.
- **Beak**: Triangle pointer at bottom.
- **Content**: Price text.
- **Wishlist Heart**: Visible if saved by user.

## States
- **Default**: White bg, text color.
- **Selected**: **Teal** (System Primary/Secondary) border/highlight.
