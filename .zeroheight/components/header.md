# Header (Navigation Bar)

**Link**: [Navigation Bar Documentation](https://zeroheight.com/8ei12wwuw/p/653285-button) (Note: Verify URL, extracted via Search)

## Usage
Placed at the top of the page. Includes the Logo and Navigation items.

## Structure
1.  **Container**: Full width, fixed at top (usually).
2.  **Logo**: Branding area (left aligned).
3.  **Navigation Items**: Links to main sections.
4.  **Scroll Button (Optional)**: Flat button on the right if content exceeds width.

## Anatomy (Navigation Item)
-   **Label**: Concise text indicating the destination.
-   **Interaction**: Clickable area around the text.

## States
-   **Selected**: Visual indicator (underline or bold).
-   **Unselected**: Default state.
-   **Hover**: Visual feedback (Web only).

## Dos & Don'ts
-   **Do**: Use concise labels.
-   **Do**: Arrange items by priority.
-   **Don't**: Overcrowd the header (use a "More" menu or scroll if necessary).
