# Rating

**Link**: [Rating Documentation](https://zeroheight.com/8ei12wwuw/p/653285-button) (Note: Verify URL, extracted via Search)

## Usage
Visual representation of evaluation scores (e.g. Hotel Reviews).
-   **Context**: Appearance in accommodation thumbnails (cards) or review sections.

## Structure
-   **Rating Score**: Numerical value (e.g. 4.8).
-   **Star/Icon**: Visual indicator (Star usually).
-   **Review Count**: Optional count (e.g. "(120)").

## Behavior
-   **Read-only**: Displaying an aggregate score.
-   **Interactive**: User inputting a score (if applicable to the context).

## Dos & Don'ts
-   **Do**: Use consistently with text scores to reinforce credibility.
-   **Don't**: Use for non-quantitative feedback.
