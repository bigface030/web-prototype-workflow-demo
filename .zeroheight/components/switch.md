# Switch (Toggle)

**Link**: [Switch Documentation](https://zeroheight.com/8ei12wwuw/p/653285-button) (Note: Verify URL, extracted via Search)

## Usage
Toggle between binary states (On/Off) with immediate effect.

## Anatomy
1.  **Track**: Background pill.
2.  **Handle**: Sliding indicator.

## States
-   **On**: Active color track, handle to right.
-   **Off**: Grey track, handle to left.
-   **Loading**: Spinning loader in handle. Used when action needs server confirmation.
-   **Disabled**: Lower opacity, non-interactive.

## Dos & Don'ts
-   **Do**: Use for immediate actions settings (e.g. "Airplane Mode").
-   **Don't**: Use for "Submit" type forms where review is needed (Use **Checkbox**).
