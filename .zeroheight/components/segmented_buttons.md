# Segmented Buttons

## Usage
linear set of two or more segments that function as mutually exclusive buttons.
- Differs from Tabs (which switch views). Segmented buttons switch **modes/states**.

## Anatomy
- **Container**: Grey background (#F5F5F5).
- **Segments**:
    - **Active**: Black bg (#000000), White text. Shadow/Elevation.
    - **Inactive**: Transparent bg, Grey text (#595959).
- **Corner Radius**: 8px.

## Behavior
- **Single Choice**: Only one active at a time.
- **Capacity**: 2-5 options recommended.

## Content
- **Label**: Short text.
- **Icon**: Optional, but must be consistent (all have icons or none).
