# Navigation Bar

## Usage
Allows users to switch between different views or categories under the same theme.

## When Not to Use
- **Sequential Content**: Do not use for steps or chapters meant to be read in order.
- **Simultaneous Viewing**: If the user needs to compare content in different tabs, do not separate them.
- **Anchor Navigation**: Use the **Tabs** component if clicking only jumps to a section on the same page.

## Structure
- Includes text labels with an active indicator (underline/highlight).

## Behavior
- **Scrollable**: Horizontal scroll enabled on mobile if items exceed width.
- **Sticky**: Can be pinned to the top of the viewport during scrolling.

## Specs
- Defines specific spacing/padding between labels to ensure touch target accessibility.
