# Drawer

## Usage
Side panel for navigation or secondary actions.
- Common: Mobile Menus, Filters, Settings.

## Specs
- **Width**: **80%** of screen width (e.g., 256px on 320px screen).
- **Header Height**: 48px.

## Navigation Levels
- **Level 1**: Main categories.
- **Level 2**: Sub-menus (slides in from right).
    - Includes "Back" button in header.

## Anatomy
- **Header**: Title + Close/Back button.
- **Section Title**: Non-clickable group header (Grey bg).
- **Items**: Clickable links (approx 48px height usually).
    - Badge: Optional (e.g., "New").
    - Chevron: Indicates sub-menu.

## Behavior
- Click Overlay to close.
- Scrollable content area.
