# Tabs

**Link**: [Tabs Documentation](https://zeroheight.com/8ei12wwuw/p/653285-button) (Note: Verify URL, extracted via Search)

## Usage
Switch between different views of information within the **same context**.

## Types
-   **Scrollable**: Width of tab based on label length. Horizontal scroll on overflow.
-   **Fixed**: Equal width for all tabs (best for few items, e.g. 2-3).

## Anatomy
1.  **Tab Item**: Clickable container.
2.  **Label**: Text description.
3.  **Indicator**: Active line/underline showing selection.
4.  **Divider**: Bottom border separating tabs from content.

## Behaviors
-   **Sticky**: Header often sticky at top during scroll.
-   **Interaction**: Click to switch content pane immediately.

## Dos & Don'ts
-   **Do**: Use for parallel content (e.g. "Details", "Reviews", "Policies").
-   **Don't**: Use for sequential steps (Use **Stepper**).
-   **Don't**: Use for anchor links on the *same* long page (Use **Anchor Menu**).
