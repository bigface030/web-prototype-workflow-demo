# Search Bar

## Usage
Primary navigation tool for finding properties/experiences.

## Typography Specs
- **Desktop/Tablet**:
    - Place/Guests: 16px Medium, Color #262626.
- **Mobile/MiniTablet**:
    - 14px Medium.

## Search Fields
1.  **Destination (目的地)**:
    - Click triggers popup with popular locations (Taipei, Taichung, etc.).
    - Includes "No limit" (不限) option.
2.  **Time Interval (時間區間)**:
    - Click triggers 'Date Picker' modal.
    - Dates range display.
3.  **Guests (人數)**:
    - Stepper input for Adults/Children/Infants.

## Behavior
- **Sticky**: Often sticky at top of results page.
- **Collapsed vs Expanded**: Mobile often shows collapsed 'pill' version until tapped.
