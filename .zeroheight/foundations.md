# Ark Design System - Foundations

## Colors
Derived from component observations:
- **Primary**: #0F7AAB (Blue/Teal variety) - Used in DatePicker Selected, etc.
- **Neutral Grey**: #F5F5F5 (Backgrounds, Disabled states).
- **Text Dark**: #262626 (Primary text).
- **Text Grey**: #595959 (Secondary text).
- **Border Default**: #D9D9D9.
- **Error/Destructive**: (Implied red for Danger buttons).

## Typography
- **Desktop Body**: 16px.
- **Mobile Body**: 14px.
- **Weights**: Regular (400), Medium (500), Bold (700).
- **Font Stack**: (Standard sans-serif stack usually implied: Roboto, Inter, or System Fonts).

## Spacing
- **Base Unit**: 4px or 8px commonly used.
- **Common Margins**: 16px (Mobile side margins, Alert padding).

## Radius
- **Buttons/Fields**: 8px (Segmented control).
- **Cards/Dialogs**: (Implied 8px or 12px based on modern standards if not specified, 16px for larger cards).

## Breakpoints
- **Mobile**: < 375px.
- **Tablet**: > 375px (approx).
- **Desktop**: Standard desktop widths.
